Text: 

Tijdens de Basiscursus Programmeren in Python bij ZB45 Makerspace leer je onder andere de structuur van een programma en het schrijven van je eigen code.

Programmeren is leuk, een creatieve bezigheid en soms ook gewoon praktisch. In deze workshop serie leer je de basis van programmeren in de taal Python.

Op steeds meer scholen wordt Python gebruikt in de Informatica en Techniek lessen. Een Python script kan bijvoorbeeld gebruikt worden om data te lezen en te ordenen, om berekeningen te maken of om bijvoorbeeld een robotje aan te sturen.

In deze cursus van 5 lessen leer je als docent onder andere wat de structuur van een programma is, hoe je bestanden kan lezen en schrijven en hoe je eigen code of code van anderen kan hergebruiken.

Uitleg wordt afgewisseld met praktische programmeer oefeningen. Na vijf weken ben je in staat om kleine applicaties te schrijven.

Start curus: donderdagavond 25 januari van 19.00 tot 21.30.
Data cursus: donderdag 25 januari, 1, 8, 15 en 22 februari

De cursus die start op donderdag 25 januari is alleen voor vrouwen.

Wil je met een aantal docenten samen op cursus, neem dan contact met op met ZB45 Makerspace om een afspraak te maken.

Meer informatie over deze cursus en het aanmeldformulier vind je via https://zb45.nl/python
